# 🇩🇪 Deutsch Flash Kart

**Almanca B2-C1 Seviye Flash Kart Uygulaması (Türkçe)**

Almanca öğrenen ve ileri seviyeye gelmek isteyen kullanıcılar için profesyonel bir flash kart uygulaması.

## 🎯 Özellikler

✅ **300+ Almanca Kelime** - B2-C1 seviyesine uygun
✅ **Kelime Kategorileri** - Fiil (Verb), İsim (Noun), Sıfat (Adjective)
✅ **İlerleme Takip** - Kaç kelime öğrendiğinizi görün
✅ **Favori Bölümü** - Sevdiğiniz kelimeleri kaydedin
✅ **Telaffuz Rehberi** - Her kelime için telaffuz yazılması
✅ **Örnek Cümleler** - Bağlamda kelime kullanımı
✅ **LocalStorage** - Verileri otomatik kaydedin
✅ **Responsive Tasarım** - Mobil ve masaüstü uyumlu
✅ **Modern UI** - Profesyonel ve kullanıcı dostu arayüz

## 🚀 Başlangıç

### Gereksinimler
- Node.js (v14 veya üstü)
- npm veya yarn

### Kurulum

1. Repository'yi klonlayın:
```bash
git clone https://github.com/erenerbs26-cmd/deutsch-flashcard.git
cd deutsch-flashcard
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Uygulamayı başlatın:
```bash
npm start
```

4. Tarayıcıda açın:
```
http://localhost:3000
```

## 📖 Nasıl Kullanılır?

### 1. Öğrenme Modu
- **Kategori Seçin**: Fiil, İsim veya Sıfat kategorilerinden seç
- **Kartı Çevir**: Almanca kelimeyi görmek için kartı tıkla
- **Türkçesini Gör**: Çeviriye ve örnek cümleye bak
- **Öğrendim Butonunu Tıkla**: Kelimeyi "Öğrenildi" olarak işaretle
- **İleri/Geri**: Sonraki veya önceki kelimeye geç

### 2. Favori Yönetimi
- **⭐ Butonu**: Kelimeyi favorilere ekle/çıkar
- **Favoriler Sekmesi**: Yalnızca favori kelimelerinizi çalışın

### 3. İlerleme Takibi
- **📊 İlerleme Sekmesi**: Kaç kelime öğrendiniz görün
- **Motivasyon Mesajları**: Başarınıza göre teşvik alın
- **İstatistikler**: Detaylı istatistikler ve grafikler
